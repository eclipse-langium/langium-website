Langium is an open source language engineering tool with first-class support for the Language Server Protocol, written in TypeScript and running in Node.js.

Learn how to [get started](/doc/getting-started/) with Langium, [what you can do with it](/doc/tutorials) and how to implement it in your project.

## Want to contribute?

Visit the [Langium repository](https://github.com/langium/langium) and take part in improving Langium.
